Agent support

Use the AI CLI you trust

Terminai does not own your model credentials or provider routing. Your configured CLI handles auth and model selection; Terminai provides the terminal wrapper, MCP server, and approval flow.

Codex

Bundled preset injects Terminai MCP tools and context into Codex.

Claude Code

Bundled preset starts Claude with a strict MCP config and Terminai system context.

Custom

Point Terminai at any CLI that can consume the MCP URL and context prompt.

Get started

Install, authenticate, launch

Install Terminai, sign in to your preferred AI CLI, then start a wrapped shell.

Homebrew

brew tap emosenkis/termin.ai https://github.com/emosenkis/termin.ai.git
brew install terminai

Source

git clone https://github.com/emosenkis/termin.ai.git
cd termin.ai
cargo build --release -p termin

Config

mkdir -p ~/.config/terminai
terminai init-config
terminai
